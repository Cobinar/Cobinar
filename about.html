    About Cobinar

    Origin &
    Vision

    Born from a single belief: that software can be both ruthlessly functional and genuinely beautiful — and that one person with enough resolve can build both at once.

    01

    The person behind it

    —

    Founder & Creator

    —

    Marc-Arthur Samuel Dalus is an independent developer and designer building the Cobernal Systems ecosystem — a suite of interconnected software products crafted under a single design philosophy: neo-minimalist precision.

    Across every product — from Binaris (a 24-language AI assistant) to Cobernal Navigatè (a custom desktop browser) — one thing remains constant: an obsessive commitment to the idea that daily tools should feel as good as they work.

    Cobinar is not a venture-backed company. It is a statement of intent — that one developer with the right standards can build software that rivals what entire teams produce.

    "I don't build products for scale, I build them for precision. The scale follows when the craft is right."

    02

    Where it started

    The Problem

    Software was either functional or beautiful. Rarely both.

    The tools that performed well were built for function, not feeling. The ones that looked good rarely worked at the depth serious users needed.

    Cobinar began as a personal refusal to accept that compromise — a collection of tools built to the standard one developer believed was possible but rarely attempted.

    How we build

    Principle 01

    Aesthetics are not optional

    Visual quality is an architectural constraint built in from the first line of code — not a polish pass applied at the end.

    Principle 02

    Language is a product feature

    Building for 24 languages natively isn't a checkbox. Every speaker deserves the same quality of experience.

    Principle 03

    PWA standards are the floor

    Installable, offline-capable, and app-fast. Progressive Web App standards are the minimum viable experience.

    Principle 04

    Built with intention

    No committee design, no corporate bloat. One developer, one vision — every decision deliberate and purposeful.
